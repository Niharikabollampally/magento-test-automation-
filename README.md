# Magento Test Automation

This project automates testing of the Magento demo website user flow.

## Tests Covered
- ✅ User Registration
- ✅ User Login
- ✅ User Logout
- ✅ Password Change

## How to Run

### In GitHub Codespaces:
1. Open this repository in Codespaces
2. Run: `npm install`
3. Run: `npm test`
